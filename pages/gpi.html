Outils

Versionning et Gestion de projet/incidents :

Git et GitHub font partie des outils principaux dont je me sert pour la gestion du patrimoine, ils me permettent de versionner mes projets et de les rendre transportable facilement d'une machine à l'autre. De plus le fonctionnalité "Issues" de GitHub me permet de faire de la gestion d'incidents facilment, de même je peux grâce à l'onglet "Projets" créé un Kanban pour gérer l'avancement de mes projets. J'utilise aussi Trello pour la gestion de projet, particulièrement en groupe ou pour ma vie personnel, pour sa facilité d'utilisation et son accessibilité. Pour les discutions instantanées et les réunions en vocal je me sert de l'application Discord.


Framework et IDE :

Pour la création de mes projets j'ai pu me servir de différents IDE (Integrated Development Environnement) tels que NetBeans, Visual Studio et PHP Storm (en période d'essai uniquement pour ce dernier). Je me suis également servi de Visual Studio Code malgré le fais qu'il ne soit pas réélement un IDE. Accompagné des outils fournis par les IDE j'ai pu profité d'outils supplémentaires fournis par des Framework, notament Laravel et Symfony en PHP.


Outils de veille

Pour effectuer ma veille j'utilise le plus souvent des réseaux sociaux, notamment LinkedIn et YouTube. Cela me permet de voir une grande diversité d'informations dans plusieurs langues avec des sources variées ainsi que de les enregistrés facilement pour pouvoir les consultés plus tard.


Technologies et Langages

Au cours de cette année scolaire j'ai pu rencontré différents langages: C#, PHP, JavaScript, HTML, CSS ... Et j'ai notamment utilisé le langage de programmation C# et le langage de programmation PHP dans mes projets. De plus j'ai pu enrichir ces projets grâce à des packages, bibliothèques, ou extensions tels que Twig, Blade, StyleCop, NewtonJsonsoft, Bootstrap, React etc...
